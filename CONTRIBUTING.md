# Contributing to Awesome SEO MCP Servers & Agent Skills

Thank you for contributing! This list aims to be the most comprehensive directory of SEO-related MCP servers, agent skills, and open source tools.

## How to Contribute

### Adding a New Entry

1. Fork this repository
2. Add your entry to the appropriate section in `README.md`
3. Follow the existing table format
4. Submit a pull request

### Entry Requirements

**MCP Servers must:**
- Be publicly accessible (open source on GitHub or listed on a public directory)
- Be related to SEO, search marketing, or web optimization
- Include: name (linked to repo), stars count, status (Active/Archived/Dead), and a brief description

**Agent Skills must:**
- Follow the SKILL.md specification
- Be publicly available on GitHub
- Include: name (linked to repo), stars count, and a brief description

**Standalone Tools must:**
- Be open source with a public repository
- Be related to SEO or search marketing
- Include: name (linked to repo), stars count, status, and description

### Quality Standards

- Verify the project is not abandoned (check last commit date)
- Mark dead/archived projects appropriately
- Keep descriptions concise (one line)
- Use accurate star counts (round to nearest significant number for 1000+)

### Reporting Issues

- Found a dead link? Open an issue
- Know a tool that's missing? Open a PR or issue
- Incorrect information? Open a PR with the fix

## Format

Use this format for new entries:

```markdown
| [project-name](https://github.com/user/repo) | 123 | Active | Brief one-line description |
```

## Code of Conduct

Be respectful and constructive. This is a community resource — help make it better for everyone.
