# Awesome SEO MCP Servers & Agent Skills [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of MCP servers, agent skills, and AI-powered tools for SEO professionals. Updated regularly.

MCP (Model Context Protocol) servers connect AI coding tools like Claude Code, Cursor, GitHub Copilot, OpenAI Codex, Windsurf, and others to SEO data and workflows. Agent Skills (SKILL.md) provide SEO capabilities directly inside AI agents.

**Why this list?** The SEO MCP ecosystem is growing fast but fragmented across dozens of directories. This is the single source of truth for every SEO-related MCP server and agent skill available today.

## Notable Gaps in the Ecosystem

These are categories with **no viable open source solution**:

| Category | What's Missing | Incumbent Price |
|----------|---------------|-----------------|
| **Content Optimization** | No open source Surfer SEO / Clearscope. ContentSwift is dead. | $89-170/mo |
| **AI Visibility Tracking** | No comprehensive GEO/AEO tracking platform. getCito barely functional. | $100-500/mo |
| **SEO A/B Testing** | No open source SearchPilot alternative. | $2,000+/mo |
| **IndexNow / Indexing API** | No MCP server or tool for instant URL indexing. | Free APIs, no tools |
| **Keyword Cannibalization** | No standalone detection tool. | Part of $99+/mo suites |
| **Log File Analysis** | No usable SEO-focused log analyzer. | $99/yr (Screaming Frog) |
| **Internal Linking** | No platform-agnostic link analysis tool. | $69+/mo (InLinks) |
| **Schema Generation MCP** | No dedicated structured data MCP server. | Free web tools exist |
| **robots.txt AI Manager** | No visual AI bot management dashboard. | Manual editing |
